The Speed Unpausing Bug

The Speed Unpausing Bug (I call it The SU Bug for short) is a bug that is caused by pausing an unpausing the game repeatedly

The bug is caused everytime the game is unpaused. When the game is unpaused the game sets the "gameActive" variable to true and

adds another set of frames into the frames that are already being played, so instead of one frame every 16ms per frame

there will be another frame in those 16ms for everytime you unpause the game.

Try this out! Press any key to start the programme and use escape to pause it then any key to unpause it again.

This can be repeated as often as your computer can handle.
